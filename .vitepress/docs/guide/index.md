# Getting started

## Installation

- Node.js version 18 or higher.
- A package manager: e.g. npm, pnpm, ...

::: code-group

```sh [npm]
$ npm install @ogs-gmbh/ngx-template-engine
```

```sh [pnpm]
$ pnpm add @ogs-gmbh/ngx-template-engine
```

```sh [yarn]
$ yarn add @ogs-gmbh/ngx-template-engine
```

```sh [bun]
$ bun add @ogs-gmbh/ngx-template-engine
```

:::

## Usage

This package offers runtime template compilation and data-binding abstractions, enabling various integration patterns.

Consult our [reference](/reference) to apply the library effectively.
